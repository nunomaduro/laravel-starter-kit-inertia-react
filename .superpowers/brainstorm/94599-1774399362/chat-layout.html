Key Layout Decisions

420px slide-over — wide enough for structured responses (tables, cards), narrow enough to keep page content visible
Context bar — always shows what page/entity the agent is aware of, builds trust
Chat / History tabs — keeps conversation list accessible without cluttering the chat view
Expand button — opens full /chat page for longer sessions, slide-over for quick queries
Input row — file attach, voice, text input, send — all at the bottom, familiar pattern
